# SellFit Price Guard — Chrome 확장 설치 안내

쿠팡 판매가를 매일 자동 수집해 SellFit 가격 추적 보드에 채우는 확장입니다.
**읽기·기록만 합니다 — 가격 결정·변경 기능은 없습니다.**

## 1. 설치 (개발자 모드 수동 로드, 약 3분)

1. 이 저장소의 `extension/` 폴더를 PC에 내려받습니다 (또는 전달받은 zip 압축 해제)
2. 크롬 주소창에 `chrome://extensions` 입력 → Enter
3. 우측 상단 **"개발자 모드"** 토글 켜기
4. 좌측 상단 **"압축해제된 확장 프로그램을 로드합니다"** 클릭
5. `extension` 폴더 선택 → 목록에 **"SellFit Price Guard"** 나타나면 완료

## 2. 최초 설정 (1회, 약 30초)

1. 크롬 우측 상단 퍼즐 아이콘 → **SellFit Price Guard** 클릭 (핀 고정 권장)
2. **SellFit 스토어 코드(6자리)** 입력 → 저장
   - 코드는 SellFit → Setup 탭에서 발급받은 6자리 숫자입니다
3. "연결 완료: (스토어명)" 표시되면 끝

## 3. 동작 방식

| 트리거 | 시점 | 동작 |
|--------|------|------|
| 정시 수집 | 매일 06:00 (크롬 실행 중일 때) | 자동 수집 |
| **캐치업** | **크롬을 켤 때마다** | 오늘 미수집이면 즉시 수집 — **아침에 크롬만 켜면 됩니다** |
| 미수집 알림 | 10:00까지 미수집 시 | 알림 1건 → 클릭하면 즉시 수집 |

수집 과정: 최소화된 창이 잠깐 열려 등록된 쿠팡 상품 페이지를 순서대로 읽고
자동으로 닫힙니다. 작업을 방해하지 않습니다.

수집 대상은 SellFit → Setup → 상품 목록에서 **"쿠팡 URL 연결"**된 상품입니다.

## 4. 확인

- 확장 아이콘 클릭 → "오늘: 수집 완료 (N건)" + 주간 수집률(N/7일)
- SellFit → Setup → **가격 추적** 탭 → 판매가·마진율·위험/주의/안전 배지 자동 반영

## 5. (선택) PC가 꺼져 있어도 아침 정시 수집 — Windows 작업 스케줄러

크롬을 매일 아침 자동 실행하려면, PowerShell(관리자)에서 1회 실행:

```powershell
schtasks /Create /TN "SellFit PriceGuard Chrome" /TR "'C:\Program Files\Google\Chrome\Application\chrome.exe'" /SC DAILY /ST 05:55 /F
```

- PC가 켜져 있으면 05:55에 크롬이 실행되고, 확장의 캐치업 트리거가 즉시 수집합니다
- 절전 모드에서 깨우려면: 작업 스케줄러 → 해당 작업 → 조건 탭 → "이 작업을 실행하기 위해 절전 모드 종료" 체크
- 해제: `schtasks /Delete /TN "SellFit PriceGuard Chrome" /F`

## 6. 문제 해결

| 증상 | 조치 |
|------|------|
| "스토어 미연결" | 팝업에서 6자리 코드 재입력 |
| "쿠팡 URL 연결된 상품 없음" | SellFit Setup → 상품 목록 → "쿠팡 URL 연결" 먼저 |
| 수집 실패 알림 | 쿠팡 페이지를 직접 한 번 열어 정상 표시되는지 확인 (로봇 확인 페이지가 떴을 수 있음) |
| 확장이 사라짐 | 개발자 모드 확장은 크롬 업데이트 후 꺼질 수 있음 — chrome://extensions에서 다시 켜기 |
