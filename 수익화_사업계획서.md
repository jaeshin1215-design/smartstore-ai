## 2단계: 유료 전환 시작 (1~2개월)

### 목표
- 첫 유료 고객 확보
- 월 ₩50만 이상 수익

### 가격 전략 (프리미엄 모델)

| 플랜 | 가격 | 기능 | 대상 |
|------|------|------|------|
| **무료** | ₩0 | 하루 5회 사용 | 체험용 |
| **스타터** | ₩9,900/월 | 하루 30회 | 초보 셀러 |
| **프로** | ₩19,900/월 | 무제한 | 활성 셀러 |
| **비즈니스** | ₩39,900/월 | 무제한 + 우선지원 | 전문 셀러 |

### 유료 전환 방법

#### Step 1: 사용 횟수 제한 추가
```
무료 사용자 → 하루 5회 제한
"더 사용하려면 업그레이드하세요" 메시지
```

#### Step 2: 결제 시스템 연결
- **토스페이먼츠** (국내 가장 쉬움, 수수료 3.3%)
- 또는 **아임포트** 활용

#### Step 3: 유료 기능 추가
무료 플랜엔 없는 기능 추가:
- 히스토리 저장 (이전에 만든 것 다시 보기)
- 대량 생성 (한 번에 여러 상품)
- 엑셀 내보내기

### 예상 수익 계산
```
스타터 30명 × ₩9,900 = ₩297,000
프로 10명 × ₩19,900 = ₩199,000
총합 = 월 약 ₩500,000
```
