# 스마트스토어 AI 도우미 전체 프로젝트 정리

작성일: 2026년 4월 14일

---

## 1. 프로젝트 개요

### 목표
Claude Code로 스마트스토어 셀러를 위한 AI 도구를 만들어
유튜브 채널 운영 + 미국 시장 진출 + 수익 창출

### 기술 스택
- **프레임워크:** Next.js + TypeScript
- **스타일:** Tailwind CSS + Pretendard 폰트
- **AI:** OpenRouter API (Claude 3 Haiku 모델)
- **배포 예정:** Vercel

## 4. 시장 분석

### 국내 시장
- 스마트스토어 판매자: 약 60만명
- 1인 셀러 비율: 70% (약 42만명)
- 경쟁 현황: 셀러툴, 플레이오토 등 (AI 도구는 거의 없음)

### 국내 대기업 경쟁 분석
| 대기업 | 도구 | 우리와 차별점 |
|--------|------|--------------|
| 네이버 | 클로바 스튜디오 | 우리는 더 빠르고 맞춤화 가능 |
| 카카오 | 카카오 i | 타겟이 다름 |

**결론:** 법적 문제 없음, 틈새 시장 충분히 존재

### 유사 성공 사례
| 서비스 | 현재 규모 | 우리와 차이 |
|--------|----------|------------|
| 셀러툴 | 연매출 수십억 | AI 없이 성공 → 우리는 AI 활용 |
| 플레이오토 | 업계 1위 | 주문 자동화 → 우리는 콘텐츠 자동화 |
| 아임웹 | 기업가치 수백억 | 쇼핑몰 제작 → 우리는 운영 도구 |

## 5. 타겟 페르소나 (확정)

### 메인 타겟: "스마트스토어 3~12개월 차 1인 셀러"

| 항목 | 내용 |
|------|------|
| 나이 | 25~45세 |
| 상품 수 | 10~100개 |
| 월 매출 | 100~500만원 |
| 하루 운영 시간 | 2~4시간 |
| 가장 큰 고충 | 시간 부족 + 전문 지식 부족 |
| 결제 의향 | ₩9,900~₩19,900 가능 |
| 시장 규모 | 약 18만명 |

### 왜 이 타겟인가?
- 초보: 결제 설득 어려움
- 전문가: 요구 수준 너무 높음
- **중간 (3~12개월): 필요성 높고 결제 의향 있음** ✅

## 6. 앱의 한계 및 개선 방향

### 현재 한계
| 기능 | 한계 | 개선 방향 |
|------|------|----------|
| 설명문 생성 | 개인 브랜드 톤 없음 | 톤 선택 기능 추가 |
| 키워드 추천 | 실제 검색량 데이터 없음 | 활용 위치 가이드 추가 |
| 가격 분석 | 경쟁사 가격 미반영 | 경쟁사 가격 입력란 추가 |

### 앞으로 추가할 기능
1. 브랜드 톤 선택 (감성적/전문적/친근함)
2. 결과물 활용 가이드
3. 경쟁사 가격 입력 후 전략 분석
4. 히스토리 저장 (유료 기능)

## 7. 수익화 전략

### 가격 플랜
| 플랜 | 가격 | 기능 |
|------|------|------|
| 무료 | ₩0 | 하루 20회 |
| 스타터 | ₩9,900/월 | 무제한 |
| 프로 | ₩29,900/월 | 무제한 + 히스토리 저장 |

### 수익 타임라인
| 시기 | 목표 | 예상 월수익 |
|------|------|------------|
| 지금 | 배포 & 무료 오픈 | ₩0 |
| 1개월 | 사용자 100명 확보 | ₩0 |
| 2개월 | 유료 전환 시작 | ₩50만~ |
| 3~6개월 | 성장 안정화 | ₩200만~ |
| 6개월~ | 스케일업 | ₩500만~ |
| 1년 | 미국 진출 준비 | ₩1,000만~ |

### 유료 전환 시점
```
사용자 100명 돌파 즉시 유료 플랜 도입
얼리버드 50명 → 평생 50% 할인으로 빠른 전환 유도
```

## 9. 국내 → 미국 진출 전략

### 순서
```
국내 출시 → 피드백 수집 → 앱 완성도 향상
→ 영어 버전 제작 → 유튜브 채널 시작
→ 미국 Amazon 셀러 타겟 → 수익 극대화
```

### 미국 시장 가격
```
국내: ₩9,900~₩29,900/월
미국: $29~$99/월 (약 5배)
```
